Own your analytics again

Google Analytics samples your data, asks visitors to accept cookies, and keeps everything on Google's servers. Insight does the opposite. You install it once, drop one script tag on your sites, and every visit lands in a database you control. No third party reads it, and there is nothing to consent to because there are no cookies.

Everything on one screen

The numbers people open an analytics tool for, plus the ones Google Analytics never showed you.

Real-time dashboard

Visitors, pageviews, bounce rate and average time, refreshed every five seconds. The number moves while you watch it.

Live world map

A globe of who is on your site right now, with the city, the page and the referrer for each visitor.

Revenue attribution

Connect Stripe and see which channels and campaigns bring paying customers, with refunds counted against the day they happened.

Funnels and retention

Track drop-off across the steps that matter, and see how many visitors come back week after week.

AI crawler tracking

See when ChatGPT, Claude, Perplexity and Google's AI bots read your pages, and which pages they read.

GA4 and Search Console

Pull Google Analytics 4 and Search Console into the same dashboard when you want the old history or the search queries.

Cookieless tracker

A small first-party script with no cookies and no fingerprinting. Nothing to disclose to visitors, nothing for a blocker to catch.

Make it yours

Drag the cards to reorder your dashboard and hide the ones you never look at. It remembers your layout on each device.

Ask your assistant for the numbers

Insight ships a read-only command line. Point Claude or a terminal at it and ask in plain words, for example "how many visitors last month and the top pages".

The token can only read. It never unlocks the dashboard, which still needs your password and a 2FA code.

Run it in a few minutes

One server, one compose file, one script tag. After that the data is yours.

1

Set three values

Copy the compose file and set your admin email, your password and a random secret.

2

Start the container

It brings its own database and serves the dashboard over HTTPS behind Cloudflare or Caddy.

3

Add one script

Paste a single script tag on each site you want to measure, and visits start showing up live.

Private by default

There are no cookies, so you skip the banner and stay clear of the consent rules that apply to Google Analytics. Visitor ids are first-party and built without fingerprinting. The data never leaves the server you chose, in the country you chose.

Questions

Is Insight really free?

Yes. The code is open source under the MIT licence. You pay only for the small server you run it on.

Is it a Google Analytics alternative?

It covers the numbers most people open Google Analytics for, and adds revenue, a live map and AI crawler tracking. You can also connect GA4 if you want the old history in the same place.

Does it use cookies?

No. The tracker is cookieless and sets nothing on the visitor device, so you do not need a cookie banner.

What do I need to self-host it?

Docker and a small server. One compose file brings up the app and the database over HTTPS.

Can it track AI crawlers?

Yes. It records visits from ChatGPT, Claude, Perplexity, Googlebot and others, per page, so you can see what the AI answers are reading.

How is the dashboard secured?

Access needs your email, your password and a 2FA code. There is no public sign-up, so nobody can create an account on your instance.
